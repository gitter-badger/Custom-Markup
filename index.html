How To Use

Direct Link

The first thing that you need to do is to link it in the <head> tag like so:

Hint: Click the code to copy it to your clipboard.

<link rel="stylesheet" type="text/css" href="https://custommarkup.ml/dist/main.min.css">

Just experiment by using the tags in the list below. Just put in text as a child in the tag.

Custom Markup is fast and lightweight. It doesn't take long to load at all. You can even use it with a css framework by putting classes in the custom made tags. This is only 846 bytes minified. Please remember to star this on github or just hit the star button in the bottom left corner.
