About

Custom Markup is fast and lightweight. It doesn't take long to load at all. This is BETTER than a CSS framework because you can use your own tags with it instead of using the tags in the CSS framework. This is only 715 bytes minified and compressed. Please remember to star this on github or just hit the star button in the bottom right corner.

Documentation

Direct Link

The first thing that you need to do is to link it in the <head> tag like so:

Hint: Click the code to copy it to your clipboard.

<link rel="stylesheet" type="text/css" href="https://custommarkup.ml/dist/cm.min.css">
